# Deskripsi

Membuat sebuah program sederhana bahasa C yang mengimplementasikan materi pada praktikum pertemuan ke-8 berupa Singular Linked List.

## Ketentuan

1. Buat 5 list mahasiswa
2. Cetak list awal
3. Hapus node posisi pertama dan terakhir
4. Cetak hasil list setelahnya
5. Tambahkan mahasiswa baru di node posisi pertama, ketiga, dan terakhir
6. Cetak hasil list setelahnya

### Challenge

- Cobalah untuk menghapus salah satu daftar di posisi tertentu (delete at position) sebagai latihan (Tidak Wajib).
