## Deskripsi

Program mengambil input sebuah bilangan bulat positif n. Dengan menggunakan konsep fungsi rekursi, program harus mencetak deret angka dari n sampai 1 secara menurun.

## Format Masukan

Sebuah bilangan bulat positif n.

## Format Keluaran

Deret angka dari n sampai 1 secara menurun.

## Contoh Masukan 1

```
7
```

## Contoh Keluaran 1

```
7 6 5 4 3 2 1
```

## Petunjuk
- Wajib gunakan fungsi rekursi.
- Dilarang menggunakan perulangan (for, while, do-while).
